# iconfont
iconfont
you can load from css.
<code>
/* フォントの定義 */
@font-face {
  font-family: 'ICONFONT';
  src: url('https://github.com/ams-design/iconfont/releases/download/1/ICONFONT.ttf') format('truetype');
  font-weight: normal;
  font-style: normal;
}

/* ifont クラスにフォントを適用 */
.ifont {
  font-family: 'ICONFONT', sans-serif;
}
</code>
